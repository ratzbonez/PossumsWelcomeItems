# Possum's Welcome Items
## Features
- Gives players configurable welcome items on their first join.
- Supports 9 welcome item slots, matching hotbar slots 1 through 9.
- Supports custom-written books
- In-game commands to set, remove, list, get, enable, disable, and reload welcome items.
- Operator-only commands.

## Commands

### /welcomeitems enable
Enable the plugin.

### /welcomeitems disable
Disable the plugin.

### /welcomeitems status
View whether welcome items are enabled and how many slots are configured.

### /welcomeitems reload
Reload the config file.

### /welcomeitems list
View all configured welcome item slots.

### /welcomeitems get
Manually give yourself the configured welcome items.

### /welcomeitems set <1-9>
Save the item in your hand to the selected welcome item slot.
